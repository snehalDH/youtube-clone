# YouTube Clone

A YouTube-inspired web application built with React.

## Tech Stack

- React 19
- JavaScript
- CSS
- Create React App

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm

### Installation

```bash
git clone https://github.com/snehalDH/youtube-clone.git
cd youtube-clone
npm install
```

### Running Locally

```bash
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

## Available Scripts

| Command | Description |
|---|---|
| `npm start` | Start development server |
| `npm test` | Run tests in watch mode |
| `npm run build` | Build for production |
